React Native Environment Setup

← Back to App Dev

Follow this guide to set up Expo, Node, and development tools for the SHPE//MAES Mobile App.

Mac Setup

Install Node.js using Homebrew and create your first Expo project.

1. Install Node.js

brew install node
node -v

2. Create a New Expo App

npx create-expo-app@latest shpe-conference-app
cd shpe-conference-app

3. Reset + Start the Project

npm run reset-project
npx expo start

4. Install Required Libraries

npx expo install expo-router react-native-safe-area-context react-native-screens expo-linking expo-constants expo-status-bar

5. Start Expo Again

npx expo start

Windows Setup

Download Node.js manually, then set up your Expo project.

1. Install Node.js

Download from the official installer:

Download Node.js 
node -v

2. Create a New Expo App

npx create-expo-app@latest shpe-conference-app
cd shpe-conference-app

3. Reset + Start the Project

npm run reset-project
npx expo start

npx expo install expo-router react-native-safe-area-context react-native-screens expo-linking expo-constants expo-status-bar

5. Start Expo Again

Command Breakdown

What each command is doing under the hood.

  • brew install node — Installs Node and npm on Mac
  • node -v — Confirms Node.js installed correctly
  • npx create-expo-app — Generates a complete Expo project template
  • npm run reset-project — Clears caches, fixes common build errors
  • npx expo start — Launches the Expo dev server and dashboard
  • expo install ... — Installs navigation, safe-area, linking, constants, status bar utilities

Running the App on Your Phone

1. Install Expo Go App

  • iOS App Store
  • Google Play Store

2. Scan the QR Code

After you run:

npx expo start

Scan the QR code from your terminal or browser.

Running the App on a Simulator

Mac (iOS Simulator)

xcode-select --install

Open Xcode → Preferences → Components → Install Simulator.

Windows (Android Emulator)

Install Android Studio and create a virtual device:

  1. Download Android Studio
  2. Open Device Manager
  3. Create a Pixel / Android 13 virtual device
  4. Run:
npx expo start --android
