App Development

About The SHPE/MAES App Dev Team

App development involves turning requirements into usable software. You’ll work on planning, UI implementation, feature development, testing, and iteration within a team setting.

Open to all experience levels. Tasks are scoped to support both new and experienced developers.

How It Works

1
Onboard

Get the tools installed and learn the basics of our stack.

2
Pick a Task

Start with beginner-friendly issues or pick a feature.

3
Build + Review

Work on a feature, and connect your code.

4
Ship

Merge, and test features in the completed application.

You’ll learn skills like
  • Reading and working with an existing codebase
  • UI building + navigation patterns
  • Debugging + testing habits
  • Talking about projects in interviews
Ways to contribute
  • Developer: build screens, components, features
  • Designer: Figma, UX improvements, UI consistency
  • Writer: documentation, guides, onboarding instructions
  • Tester: test flows, report bugs, improve quality

How to Join

We’ll get you onboarded

  1. Come to an App Dev meeting (or reach out by email).
  2. We’ll help you set up your environment + explain the codebase.
  3. You’ll pick a starter task (or join a feature team).
  4. You’ll ship your first contribution and keep progressing.
