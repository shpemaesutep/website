New Member Setup

  1. Install & Verify Git
    Step B — Install Git (if needed):
    • Mac:
      • Run in Terminal:
        • xcode-select --install
      • This installs Apple Command Line Tools (includes Git).
      • After install, run git --version again to confirm.
    • Windows:
      • Go to: https://git-scm.com/download/win
      • Download and run the installer
      • Keep default settings during installation
      • Restart terminal after install
      • Run git --version to confirm installation

  1. Open VS Code
  2. In VS Code, go to File → Open Folder and select your Desktop folder.
  3. Open the VS Code terminal: View → Terminal
  4. Clone the GitHub repository:
git clone https://github.com/shpemaesutep/appdev.git
  1. Go into the project folder:
cd appdev
Make sure your terminal path ends in appdev. Finish the rest of your setup inside this folder.

Important: Run all commands below inside the appdev folder.
cd appdev
0. Install & Verify Node.js (Includes npm)

Node.js includes npm automatically. Install it first if you don’t already have it.

Mac:
  • If you use Homebrew, run:
brew install node
  • If you don’t have Homebrew, download Node.js (LTS) from:
https://nodejs.org
Windows:
  • Download the LTS installer from:
https://nodejs.org
  • Run the installer and keep default settings
  • Restart your terminal after installation
Verify Installation

Run these in your terminal:

node -v
npm -v
If either command says command not found or is not recognized, restart your terminal and try again.
1. Install dependencies (inside the appdev folder)
npm install
2. Install required Expo libraries
npx expo install expo-router react-native-safe-area-context react-native-screens expo-linking expo-constants expo-status-bar
3. Start the app (Expo dev server)
npx expo start

Run on:

  • Phone: install Expo Go, then scan the QR code
  • iOS Simulator (Mac): install Xcode + iOS Simulator (Xcode → Settings/Preferences → Components)
  • Android Emulator (Windows): install Android Studio → Device Manager → create a virtual device
If Expo gets stuck or errors persist, try clearing cache:
npx expo start -c

Only install this if you’re helping generate builds (iOS/Android).

npm install -g eas-cli
eas --version
